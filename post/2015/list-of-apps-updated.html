List of Apps (Updated)

Again that time of the year where Windows’s being reinstalled, so once again I’m jotting down a list of apps I normally install right away, slightly updated versus last year.

(This list last updated in May 2016.)

Web

  • Skype is back. The web version was not robust enough.
  • Chrome. Don’t forget the Chrome Remote Desktop extension.
  • Dropbox
  • Google Drive, though I’ve been using InSync lately as it works better if you have many files/folders.
  • FileZilla
  • qBittorrent as a replacement for Tixati, which I was using before (and Miniget before that).

General

  • Microsoft Office
  • 7-Zip
  • Cmder makes the Windows command line ten times better. Comes with Clink. Replaces Putty as ssh is included.
  • Everything to locate files and folders by name instantly.
  • f.lux
  • Paint.NET
  • VirtualBox
  • Spotify
  • VLC
  • Steam
  • Fonts: Open Sans especially.

Data Science

  • Anaconda Python Distribution
  • Disco
  • GraphViz
  • R and RStudio

Coding and Editing

  • Eclipse
  • GitHub Desktop
  • LyX
  • Sublime Text. Don’t forget to install Package Manager with SFTP.
  • WinEdt
  • PyCharm. New on the list, could not do without anymore for larger projects.
