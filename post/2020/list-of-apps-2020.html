List of Apps (2020)

It’s time to setup a new machine, so once again I’m jotting down a list of apps I generally install right away, slightly updated again.

  • Chrome
  • Google Drive
  • FileZilla
  • Microsoft Office (including Skype for Business)
  • Microsoft Teams
  • 7-Zip
  • Everything
  • f.lux
  • Paint.NET
  • Sumatra PDF and Adobe Reader for those annoying edge cases where Sumatra doesn’t work
  • Fonts: Open Sans and Fira Code especially
  • OBS
  • VideoPad: bare bones video editor which is enough for me
  • Anaconda Python Distribution: although Python 3 has good venv support these days
  • VSCode
  • GitHub Desktop
  • Spotify
  • Steam

  • Dropbox
  • VirtualBox

  • Epic Store

  • Gimp
  • QGIS
  • GraphViz
  • R and RStudio
  • MSYS2
  • qBittorrent
  • WinEdt
  • Eclipse
  • LyX

Portable:

  • AltDrag
  • Audacity
  • AltDrag
  • BareGrep
  • DB Browser for SQLite
  • Joplin
  • Geek Uninstaller
  • ScreenToGif
