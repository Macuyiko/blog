List of Apps (2019)

It’s time to setup a new laptop, so once again I’m jotting down a list of apps I normally install right away, slightly updated again.

Web

  • Chrome
  • Google Drive
  • FileZilla

Optional:

  • qBittorrent install once required
  • Dropbox to be replaced with Google Drive completely, after their 3 device limit
  • Skype is avoided, use web version if possible

General

  • Microsoft Office
  • 7-Zip
  • Everything to locate files and folders by name instantly
  • f.lux
  • Paint.NET
  • Sumatra PDF and Adobe Reader for those annoying edge cases where Sumatra doesn’t work
  • Fonts: Open Sans and Fira Code especially
  • Putty
  • OBS
  • VideoPad
  • BareGrep

Optional:

  • VirtualBox
  • Spotify
  • VLC
  • Steam
  • GOG Galaxy
  • Gimp
  • Geek Uninstaller to clean up Windows Apps cruft
  • SQLite DB Browser

Data Science

  • Anaconda Python Distribution
  • GraphViz
  • R and RStudio preferably through Anaconda
  • MSYS2

Coding and Editing

  • VSCode
  • GitHub Desktop
  • WinEdt
  • Eclipse
  • LyX
